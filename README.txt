SITIO WEB JR AUTOPRESS

Archivos principales:
- index.html: estructura del sitio.
- css/styles.css: estilos listos para usar.
- scss/styles.scss: archivo fuente en Sass/SCSS.
- js/main.js: menú móvil, animaciones, botón superior y validación del formulario.
- assets/img/logo-jr-autopress.png: logotipo utilizado.

Para abrirlo:
1. Descomprime la carpeta.
2. Abre index.html en tu navegador.

Para publicarlo:
Puedes subir toda la carpeta a Netlify, GitHub Pages, Hostinger, cPanel o cualquier alojamiento web.

Pendientes de personalización:
- Teléfono
- Correo
- Dirección
- Catálogo real de productos o servicios
- Enlace de WhatsApp

Nota: La hoja CSS ya está compilada. No necesitas instalar Sass para usar el sitio. El archivo SCSS se incluye por si deseas editarlo y volverlo a compilar.
